2 Como importar no Elementor
Siga o passo a passo após baixar o arquivo JSON.
1
Crie uma nova página no WordPress
Páginas → Adicionar nova → coloque um título → clique em "Editar com Elementor"
2
Abra o painel de Templates
No editor Elementor, clique no ícone de pasta (⊞ Adicionar Template) no centro da tela ou no ícone de pasta no painel esquerdo.
3
Importe o arquivo JSON
Aba "Meus Templates" → clique no ícone de upload (seta para cima) no canto superior direito → selecione o arquivo .json baixado.
elementor-template.json
4
Insira o template na página
Após importar, o template aparece na lista. Clique em "Inserir" para adicionar à página.
5
Cole o CSS personalizado
Elementor → Configurações do Site → CSS Personalizado → cole o conteúdo do arquivo style.css para restaurar os estilos visuais.
6
Adicione as imagens
Vá em WordPress → Mídia → Adicionar novo. Após upload, copie a URL de cada imagem e substitua nos widgets de imagem do Elementor.
